# browsertabs (browsertabs)

## purpose

Shows live state of tabs connected to the browser shared worker.
Helps to understand how quickly your browser reduces performance on inactive tabs.
Keeps track of tab visibility.
Nominates the oldest active tab as primary.
Keeps track of connection times and pings.
